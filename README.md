# 🛤️ kalauz

## 📖 About

A program for
1. **making** MÁV's and GySEV's internal, **dirty "databases"** of speed restrictions **computer-processable**
2. **storing** that data in a **mySQL-database**
3. **visualizing** it on a **map.**
